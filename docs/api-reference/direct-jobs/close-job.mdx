## Endpoint

```
POST /api/v1/companies/me/jobs/{jobId}/close
```

**Base URL**: `http://localhost:3000/api/v1`

<Note>This endpoint requires **authentication** and **Company role**.</Note>

## Authentication

Include the access token in the Authorization header:

```bash
Authorization: Bearer YOUR_ACCESS_TOKEN
```

## Description

Close a job posting. Closed jobs are no longer accepting applications. This action is final - you cannot reopen a closed job.

## Path Parameters

<ParamField name="jobId" type="string" required>
  The UUID of the job to close.
</ParamField>

## Notes

<Info>Only jobs with status `PUBLISHED` or `PAUSED` can be closed.</Info>

<Warning>
  **Final Action**: Closed jobs cannot be reopened. If you need to hire again, create a new job
  posting.
</Warning>
