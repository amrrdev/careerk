---
title: 'Get Profile By ID'
api: 'GET /api/v1/job-seekers/:id'
description: 'Retrieve a specific job seeker public profile by their unique identifier'
---

## Endpoint

```
GET /api/v1/job-seekers/:id
```

**Base URL**: `http://localhost:3000/api/v1`

<Note>This endpoint is **public** and does not require authentication.</Note>

## Path Parameters

<ParamField path="id" type="string" required>
  The unique identifier (UUID) of the job seeker profile to retrieve.
</ParamField>

## Request Example

```bash
GET /api/v1/job-seekers/fe721809-a9da-4301-a8fe-9d3373dced00
```

## Error Responses

### 400 Bad Request - Invalid ID Format

```json
{
  "statusCode": 400,
  "message": "Invalid UUID",
  "error": "Bad Request"
}
```

### 404 Not Found - Profile Not Found

```json
{
  "statusCode": 404,
  "message": "Job seeker profile not found",
  "error": "Not Found"
}
```

## Notes

<Info>
  **Public Profile**: This endpoint returns public profile information. It includes education, work
  experience, and skills. Some sensitive data like phone and expected salary may be hidden based on
  the profile owner's privacy settings.
</Info>

<Tip>
  **View Other Profiles**: Use this endpoint to view other job seekers' public profiles when
  researching candidates or networking.
</Tip>
