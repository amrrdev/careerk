---
title: 'Get Profile By ID'
api: 'GET /api/v1/job-seekers/:id'
description: 'Retrieve a specific job seeker\'s public profile by their unique identifier'
---

## Endpoint

```
GET /api/v1/job-seekers/:id
```

**Base URL**: `http://localhost:3000/api/v1`

<Note>This endpoint is **public** and does not require authentication.</Note>

## Path Parameters

<ParamField path="id" type="string" required>
  The job seeker's unique identifier (UUID).
</ParamField>

## Request Example

```bash
GET /api/v1/job-seekers/a0000000-0000-0000-0000-000000000004
```

## Error Responses

### 404 Not Found

Returned when the job seeker profile doesn't exist.

```json
{
  "statusCode": 404,
  "message": "Job seeker profile not found",
  "error": "Not Found"
}
```

### 400 Bad Request - Invalid ID Format

```json
{
  "statusCode": 400,
  "message": "Invalid UUID format",
  "error": "Bad Request"
}
```

### 500 Internal Server Error

```json
{
  "statusCode": 500,
  "message": "Failed to retrieve profile",
  "error": "Internal Server Error"
}
```

## Notes

<Info>
  **Public Profile Access**: This endpoint returns detailed profile information including education,
  work experience, and skills. It's designed for companies viewing job seeker profiles.
</Info>

<Tip>
  **Contact Information**: The profile includes contact information (cvEmail, phone) that companies
  can use to reach out to candidates.
</Tip>
