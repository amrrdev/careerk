---
title: 'Get Company By ID'
api: 'GET /api/v1/companies/:id'
description: 'Retrieve a specific company profile by ID'
---

## Endpoint

```
GET /api/v1/companies/:id
```

**Base URL**: `http://localhost:3000/api/v1`

<Note>This endpoint is **public** and does not require authentication.</Note>

## Path Parameters

<ParamField path="id" type="string" required>
  The unique identifier of the company (UUID).
</ParamField>
