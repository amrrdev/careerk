---
title: 'Get My Company Profile'
api: 'GET /api/v1/companies/me'
description: 'Retrieve the authenticated company profile'
---

## Endpoint

```
GET /api/v1/companies/me
```

**Base URL**: `http://localhost:3000/api/v1`

<Note>This endpoint requires **Company authentication**.</Note>

## Headers

| Header        | Value                 |
| ------------- | --------------------- |
| Authorization | Bearer {access_token} |

## Error Responses

### 401 Unauthorized

```json
{
  "success": false,
  "error": {
    "message": "Unauthorized",
    "statusCode": 401,
    "timestamp": "2026-04-23T19:20:00.000Z",
    "path": "/companies/me",
    "method": "GET",
    "details": "Unauthorized"
  }
}
```
