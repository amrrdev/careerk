## Endpoint

```
DELETE /api/v1/job-seekers/me/applications/:id
```

**Base URL**: `http://localhost:3000/api/v1`

<Note>This endpoint requires **Job Seeker authentication**.</Note>

## Headers

| Header        | Value                 |
| ------------- | --------------------- |
| Authorization | Bearer {access_token} |

## Path Parameters

<ParamField path="id" type="string" required>
  The unique identifier of the application to withdraw (UUID).
</ParamField>

## Description

This endpoint withdraws the application from a job. The application status will be changed to `WITHDRAWN`. This action cannot be undone.

## Response

```json
{
  "success": true,
  "data": {},
  "message": "Application withdrawn successfully",
  "meta": {
    "timestamp": "2026-04-23T19:30:00.000Z",

    "method": "DELETE"
  }
}
```
