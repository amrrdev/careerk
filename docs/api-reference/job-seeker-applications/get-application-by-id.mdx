---
title: 'Get Application By ID'
api: 'GET /api/v1/job-seekers/me/applications/:id'
description: 'Retrieve a specific application by ID'
---

## Endpoint

```
GET /api/v1/job-seekers/me/applications/:id
```

**Base URL**: `http://localhost:3000/api/v1`

<Note>This endpoint requires **Job Seeker authentication**.</Note>

## Headers

| Header        | Value                 |
| ------------- | --------------------- |
| Authorization | Bearer {access_token} |

## Path Parameters

<ParamField path="id" type="string" required>
  The unique identifier of the application (UUID).
</ParamField>
