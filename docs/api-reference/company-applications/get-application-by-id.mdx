---
title: 'Get Application By ID'
api: 'GET /api/v1/companies/me/applications/:id'
description: 'Retrieve detailed information about a specific application'
---

## Endpoint

```
GET /api/v1/companies/me/applications/:id
```

**Base URL**: `http://localhost:3000/api/v1`

<Note>This endpoint requires **Company authentication**.</Note>

## Headers

| Header        | Value                 |
| ------------- | --------------------- |
| Authorization | Bearer {access_token} |

## Path Parameters

<ParamField path="id" type="string" required>
  The unique identifier of the application (UUID).
</ParamField>
