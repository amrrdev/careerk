---

## Endpoint

```
GET /api/v1/job-seekers/me/work-experiences/:expId
```

**Base URL**: `http://localhost:3000/api/v1`

<Note>This endpoint requires **authentication** and **Job Seeker role**.</Note>

## URL Parameters

| Parameter | Type   | Required | Description            |
| --------- | ------ | -------- | ---------------------- |
| `expId`   | string | Yes      | The work experience ID |

## Authentication

Include the access token in the Authorization header:

```bash
Authorization: Bearer YOUR_ACCESS_TOKEN
```

## Description

Retrieve a specific work experience entry by its ID. Only returns work experiences belonging to the authenticated job seeker.

## Response

### Success Response (200 OK)

```json
{
  "success": true,
  "data": {
    "id": "exp-123-uuid",
    "companyName": "Tech Corp",
    "jobTitle": "Senior Software Engineer",
    "description": "Led development of microservices architecture",
    "location": "Cairo, Egypt",
    "startDate": "2023-01-15T00:00:00.000Z",
    "endDate": null,
    "isCurrent": true,
    "jobSeekerId": "js-456-uuid",
    "createdAt": "2024-01-10T10:30:00.000Z",
    "updatedAt": "2024-01-10T10:30:00.000Z"
  },
  "message": "Success",
  "meta": {
    "timestamp": "2026-02-18T10:00:00.000Z",
    "path": "/job-seekers/me/work-experiences/exp-123-uuid",
    "method": "GET"
  }
}
```

### Response Fields

<ResponseField name="id" type="string">
  Unique identifier for the work experience entry.
</ResponseField>

<ResponseField name="companyName" type="string">
  Name of the company.
</ResponseField>

<ResponseField name="jobTitle" type="string">
  Title of the position.
</ResponseField>

<ResponseField name="description" type="string | null">
  Job description and responsibilities.
</ResponseField>

<ResponseField name="location" type="string | null">
  Work location.
</ResponseField>

<ResponseField name="startDate" type="string">
  Start date (ISO 8601 format).
</ResponseField>

<ResponseField name="endDate" type="string | null">
  End date (ISO 8601 format). Null if currently working.
</ResponseField>

<ResponseField name="isCurrent" type="boolean">
  Whether this is the current job.
</ResponseField>

## Error Responses

### 401 Unauthorized - Not Authenticated

```json
{
  "success": false,
  "error": {
    "message": "Unauthorized",
    "statusCode": 401,
    "timestamp": "2026-02-18T10:00:00.000Z",
    "path": "/job-seekers/me/work-experiences/exp-123-uuid",
    "method": "GET"
  }
}
```


### 404 Not Found

Returned when the work experience doesn't exist or doesn't belong to the user.

```json
{
  "success": false,
  "error": {
    "message": "Work experience not found",
    "statusCode": 404,
    "timestamp": "2026-02-18T10:00:00.000Z",
    "path": "/job-seekers/me/work-experiences/exp-123-uuid",
    "method": "GET"
  }
}
```

## Example Request

```bash
curl -X GET http://localhost:3000/api/v1/job-seekers/me/work-experiences/exp-123-uuid \
  -H "Authorization: Bearer YOUR_ACCESS_TOKEN"
```
